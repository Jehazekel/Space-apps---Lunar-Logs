The |UNIT| ...Lunar Logs is a web app designed to allow simultaneous collaboration in real time between EVA crewmembers and those on earth. The user interface seeks to be simple and effective and allows easy access to post-mission operation debriefs and scientific research so that it may serve as an educational platform as well.
